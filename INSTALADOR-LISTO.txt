PRÓXIMO PASO: COMPILAR EL INSTALADOR

1. Descarga Inno Setup (si no lo tienes):
   https://jrsoftware.org/isdl.php

2. Abre Inno Setup Compiler

3. File > Open > Selecciona:
   E:\prueba\setup.iss

4. Build > Compile (o presiona F9)

5. Espera 1-2 minutos

6. Se creará:
   E:\prueba\PITHY-Installer-v1.0.exe

═══════════════════════════════════════════════════════════

LO QUE HACE EL INSTALADOR:

El archivo .exe incluirá:
  • Todos los archivos de PITHY (excepto node_modules, .next)
  • Interfaz gráfica profesional
  • Licencia en español
  • Descarga e instalación automática de:
    - Node.js v20.11.0
    - Ollama AI
    - Modelo qwen2.5:3b (2GB)
    - Dependencias npm
  • Creación de iconos en escritorio:
    - INICIAR PITHY
    - DETENER PITHY
    - PANEL ADMIN PITHY
    - CARPETA PITHY
  • Desinstalador integrado

═══════════════════════════════════════════════════════════

DISTRIBUCIÓN:

Opción 1 - Internet:
  1. Sube PITHY-Installer-v1.0.exe a Google Drive/Dropbox
  2. Comparte el link
  3. Cliente descarga e instala

Opción 2 - USB:
  1. Copia el .exe a USB
  2. Llevas al restaurante
  3. Doble clic e instala

═══════════════════════════════════════════════════════════

INSTALACIÓN PARA EL CLIENTE:

1. Doble clic en PITHY-Installer-v1.0.exe
2. Acepta licencia
3. Elige ubicación (default: C:\Program Files\PITHY)
4. Click "Instalar"
5. Espera 30-60 minutos
6. Configura .env.local
7. Usa PITHY desde iconos del escritorio
